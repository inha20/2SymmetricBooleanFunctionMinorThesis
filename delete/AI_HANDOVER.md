## Research Goal

The core purpose of this project is not merely to classify visual Karnaugh-map patterns.

The deeper objective is:

- Understanding why specific visual patterns appear in Karnaugh maps.
- Interpreting these patterns through Boolean-space structure.
- Explaining visual phenomena using Hamming Weight Layer structures.
- Investigating whether Karnaugh-map patterns can be understood as projections of higher-dimensional Boolean-space structures.

---

## Important Concepts

### Layer Structure

Boolean states are grouped by Hamming Weight:

- L0
- L1
- L2
- L3
- L4

Current hypothesis:

Visual patterns are generated from the interaction between:

1. Layer structure
2. Karnaugh-map arrangement

rather than from arbitrary cell placement.
