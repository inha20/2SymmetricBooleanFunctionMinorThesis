3. 시각 패턴의 정의 3.1 시각적 대칭 

본 연구에서는 카르노맵에 나타나는 구조를 분석하기 위하여 시각적 대칭(Visual Symmetry)의 개념을 정의한다.

시각적 대칭이란 특정 축에 대하여 카르노맵을 반사하였을 때 대응되는 위치의 값이 동일하게 유지되는 성질을 의미한다.

예를 들어 세로축 대칭은

F(a,b)=F(a,5-b)

가로축 대칭은

F(a,b)=F(5-a,b)

주대각선 대칭은

F(a,b)=F(b,a)

로 표현할 수 있다.

본 연구에서는 이러한 관계가 성립할 때 해당 축에 대한 시각적 대칭이 존재한다고 정의한다.

3.2 시각적 동일 

시각적 동일(Visual Repetition)은 대칭과 구별되는 개념이다.

시각적 동일이란 특정 부분 패턴이 반복적으로 나타나는 현상을 의미한다.

예를 들어 XOR 체커보드 패턴에서는 동일한 구조가 평면 전체에 반복적으로 나타난다.

시각적 대칭이 반사 관계를 의미한다면, 시각적 동일은 반복 관계를 의미한다.

따라서 두 개념은 독립적으로 존재할 수 있다.

3.3 체커보드 패턴 

체커보드 패턴(Checkerboard Pattern)은 인접한 셀의 값이 항상 서로 반대가 되는 패턴을 의미한다.

예를 들면 다음과 같다.

0101

1010

0101

1010

4변수 XOR 함수를 Gray Code 배열(0132)로 표현하면 이러한 구조가 나타난다.

Gray Code 배열에서는 인접 셀이 정확히 하나의 변수만 다르다.

따라서 인접 이동 시 Hamming Weight의 홀짝성이 반드시 변하게 된다.

XOR 함수는 홀수 Weight 층만 선택하므로 인접 셀의 출력값이 항상 반전된다.

그 결과 체커보드 패턴이 형성된다.

본 연구에서는 이를 인접성 기반 패턴(adjacency-based pattern)으로 분류한다.

3.4 이중대각 패턴 

일반 이진 배열(0123)에서 XOR 함수를 표현하면 다음과 같은 패턴이 나타난다.

0110

1001

1001

0110

이 패턴은 주대각선과 부대각선에 대한 대칭성을 동시에 가진다.

본 연구에서는 이를 이중대각 패턴(Double-Diagonal Pattern)이라 정의한다.

체커보드 패턴이 인접성 구조를 강조한다면, 이중대각 패턴은 XOR 함수의 대칭 구조를 강조하는 표현이라 볼 수 있다.

3.5 고리 패턴 

Exactly-2 함수는 Weight가 정확히 2인 상태만 선택하는 함수이다.

4변수 Boolean 공간에서 Weight=2 상태는 총 6개 존재한다.

이를 Gray Code 기반 0132 배열에 배치하면 선택된 상태들이 띠(band) 형태의 연결 구조를 형성한다.

인간은 이러한 연결 구조를 하나의 폐곡선 또는 고리와 유사한 형태로 인식할 수 있다.

본 연구에서는 이를 고리 패턴(Ring Pattern)이라 정의한다.

고리 패턴은 단순히 Weight=2 상태가 가장 많은 층이라는 사실만으로 설명되지 않는다.

오히려 Weight=2 Layer의 구조와 Gray Code 배열의 인접성 보존 성질이 결합되어 나타나는 시각적 현상으로 해석할 수 있다.

3.6 시각 패턴 분류 체계 

본 연구에서 관찰된 주요 시각 패턴은 다음과 같이 분류할 수 있다.

패턴 대표 함수 점 패턴 Exactly-0, Exactly-4 모서리 패턴 Exactly-1, Exactly-3 고리 패턴 Exactly-2 체커보드 패턴 XOR, XNOR (0132 배열) 이중대각 패턴 XOR, XNOR (0123 배열) 

이 분류는 논리식이 아니라 카르노맵 위에 나타나는 시각적 구조를 기준으로 수행한 것이다.

이후 장에서는 각 패턴이 Boolean 공간의 Layer 구조와 어떤 관계를 가지는지 분석한다.

4. 완전대칭함수의 시각 패턴 분석 4.1 분석 대상 

본 연구에서는 입력 변수의 순서에 영향을 받지 않는 완전대칭함수(Symmetric Boolean Function)를 대상으로 시각 패턴을 분석하였다.

특히 Hamming Weight만으로 출력이 결정되는 다음 함수들을 분석 대상으로 선정하였다.

Exactly-0 Exactly-1 Exactly-2 Exactly-3 Exactly-4 XOR XNOR 

이 함수들은 모두 Weight 층 구조와 직접적으로 연결되며, Layer 기반 해석이 가능하다.

4.2 Exactly-0 함수 

Exactly-0 함수는 모든 입력이 0인 경우에만 출력이 1이 된다.

즉,

[ L_0 ]

만을 선택한다.

선택되는 상태는 단 하나뿐이므로 카르노맵에서는 하나의 점으로 표현된다.

본 연구에서는 이를 점 패턴(Point Pattern)으로 분류한다.

4.3 Exactly-1 함수 

Exactly-1 함수는 입력 중 정확히 하나만 1인 경우 출력이 1이 된다.

즉,

[ L_1 ]

을 선택한다.

Weight=1 상태는 총 4개 존재하며 카르노맵에서는 서로 떨어진 위치에 배치된다.

그 결과 네 개의 점이 모서리 방향으로 분산된 구조가 나타난다.

본 연구에서는 이를 모서리 패턴(Corner Pattern)으로 분류한다.

4.4 Exactly-2 함수 

Exactly-2 함수는 Weight가 정확히 2인 상태만 선택한다.

즉,

[ L_2 ]

를 선택한다.

Weight=2 상태는 총 6개 존재하며 4변수 Boolean 공간에서 가장 큰 Layer를 형성한다.

그러나 고리 패턴의 원인이 단순히 상태 수가 많기 때문이라고 보기는 어렵다.

Weight=2 상태들을 Gray Code 기반 0132 배열에 배치하면 선택된 셀들이 연결된 띠 형태를 형성한다.

인간은 이 구조를 하나의 폐곡선 또는 고리와 유사한 형태로 인식할 수 있다.

따라서 Exactly-2의 고리 패턴은

Weight=2 Layer의 구조 Gray Code 배열의 인접성 보존 성질 

이 결합된 결과로 해석할 수 있다.

4.5 Exactly-3 함수 

Exactly-3 함수는 Weight가 정확히 3인 상태를 선택한다.

즉,

[ L_3 ]

을 선택한다.

Weight=3 상태 수는 Weight=1 상태 수와 동일하다.

또한 카르노맵에서는 Exactly-1과 동일한 시각 구조를 형성한다.

따라서 Exactly-1과 Exactly-3은 서로 다른 Layer를 선택하지만 동일한 모서리 패턴으로 분류할 수 있다.

4.6 Exactly-4 함수 

Exactly-4 함수는 모든 입력이 1인 경우에만 출력이 1이 된다.

즉,

[ L_4 ]

만을 선택한다.

선택되는 상태 수는 1개이며 Exactly-0과 동일한 점 패턴을 형성한다.

따라서 Exactly-0과 Exactly-4는 서로 다른 Layer에 대응하지만 동일한 시각 패턴으로 분류된다.

4.7 XOR 함수 

XOR 함수는 Weight가 홀수인 상태에서 출력이 1이 된다.

즉,

[ L_1 \cup L_3 ]

을 선택한다.

0132 배열에서는 체커보드 패턴이 나타난다.

반면 0123 배열에서는 이중대각 패턴이 나타난다.

이는 동일한 Layer 구조가 배열 방식에 따라 서로 다른 시각 패턴으로 표현될 수 있음을 보여준다.

그러나 두 경우 모두 동일한 홀수 Layer 선택 구조를 기반으로 한다는 점에서, 패턴의 근본 원인은 Layer 구조에 있다고 해석할 수 있다.

4.8 XNOR 함수 

XNOR 함수는 Weight가 짝수인 상태에서 출력이 1이 된다.

즉,

[ L_0 \cup L_2 \cup L_4 ]

를 선택한다.

이는 XOR 함수의 보수 구조에 해당한다.

0132 배열에서는 XOR의 반전된 체커보드 패턴이 나타난다.

0123 배열에서는 XOR과 대응되는 이중대각 패턴이 나타난다.

따라서 XNOR 역시 Layer 선택 구조와 배열 방식의 상호작용으로 시각 패턴이 형성된다고 해석할 수 있다.

4.9 분석 결과 요약 

분석 결과 완전대칭함수의 시각 패턴은 다음과 같이 정리할 수 있다.

함수 선택 Layer 시각 패턴 Exactly-0 L₀ 점 패턴 Exactly-1 L₁ 모서리 패턴 Exactly-2 L₂ 고리 패턴 Exactly-3 L₃ 모서리 패턴 Exactly-4 L₄ 점 패턴 XOR L₁ ∪ L₃ 체커보드/이중대각 XNOR L₀ ∪ L₂ ∪ L₄ 체커보드/이중대각 

이 결과는 카르노맵의 시각 패턴이 단순한 배열 결과가 아니라 Layer 구조와 표현 방식의 상호작용을 반영한다는 사실을 시사한다.

5. XOR/XNOR 패턴의 구조적 해석 5.1 XOR 함수와 Layer 구조 

4변수 XOR 함수는 다음과 같이 정의된다.

[ A \oplus B \oplus C \oplus D ]

XOR 함수는 입력 상태에 포함된 1의 개수가 홀수일 때 출력이 1이 된다.

즉 XOR 함수는 다음 Layer를 선택한다.

[ L_1 \cup L_3 ]

반면

[ L_0,;L_2,;L_4 ]

는 선택되지 않는다.

따라서 XOR 함수는 Boolean 공간의 홀수 Weight 층을 선택하는 함수로 해석할 수 있다.

5.2 Gray Code 배열의 인접성 보존 

4변수 카르노맵은 일반적으로 Gray Code 배열을 사용한다.

행과 열은

00, 01, 11, 10

순서로 배열된다.

Gray Code의 핵심 성질은 인접한 두 상태가 정확히 하나의 비트만 다르다는 점이다.

즉 인접 상태 사이의 Hamming Distance는 항상 1이다.

따라서 카르노맵에서 상하좌우로 한 칸 이동하는 것은 Boolean 공간에서 하나의 변수만 변경하는 것과 대응된다.

본 연구에서는 이를 Gray Code 배열의 인접성 보존 성질이라 부른다.

5.3 인접 이동과 Weight 변화 

Boolean 공간에서 하나의 비트가 변경되면 Hamming Weight는 반드시 1만큼 증가하거나 감소한다.

즉

[ L_0 \leftrightarrow L_1 ]

[ L_1 \leftrightarrow L_2 ]

[ L_2 \leftrightarrow L_3 ]

[ L_3 \leftrightarrow L_4 ]

사이에서만 이동할 수 있다.

따라서 인접한 두 상태는 항상 Weight의 홀짝성이 반대이다.

이 성질은 XOR 체커보드 패턴 형성의 핵심 원인이 된다.

5.4 XOR 체커보드 패턴의 형성 

XOR 함수는 홀수 Layer만 선택한다.

[ L_1 \cup L_3 ]

Gray Code 배열에서는 인접 상태가 항상 한 비트만 다르다.

따라서 인접 이동 시 Weight의 홀짝성이 반드시 반전된다.

결과적으로 인접한 두 셀은 항상 서로 다른 출력값을 가진다.

즉

[ 0 \leftrightarrow 1 ]

관계가 모든 인접 셀에서 성립한다.

그 결과 XOR 함수는 체커보드 패턴을 형성하게 된다.

따라서 체커보드 패턴은 단순한 시각적 우연이 아니라 다음 구조의 결과로 해석할 수 있다.

XOR

↓

홀수 Layer 선택

↓

Gray Code 인접성

↓

인접 셀 출력 반전

↓

체커보드 패턴

5.5 XNOR 함수의 해석 

XNOR 함수는 XOR 함수의 보수이다.

따라서

[ L_0 \cup L_2 \cup L_4 ]

를 선택한다.

즉 짝수 Weight 층만 선택하는 함수로 해석할 수 있다.

Gray Code 배열에서는 인접 상태의 Weight 홀짝성이 항상 반대가 되므로 인접 셀의 출력 역시 항상 반전된다.

따라서 XNOR 함수 역시 체커보드 패턴을 형성한다.

XNOR 체커보드는 XOR 체커보드의 반전 형태로 이해할 수 있다.

5.6 체커보드 패턴은 왜 특별한가 

본 연구에서 분석한 완전대칭함수들은 모두 Layer 구조로 표현할 수 있었다.

그러나 모든 함수가 체커보드 패턴을 형성하지는 않았다.

함수 선택 Layer 대표 패턴 Exactly-0 L₀ 점 Exactly-1 L₁ 모서리 Exactly-2 L₂ 고리 Exactly-3 L₃ 모서리 Exactly-4 L₄ 점 XOR L₁∪L₃ 체커보드 XNOR L₀∪L₂∪L₄ 체커보드 

체커보드 패턴은 완전대칭함수의 일반적 특징이 아니다.

체커보드는 홀수 Layer 전체 또는 짝수 Layer 전체를 선택하는 패리티(parity) 구조에서 나타나는 특수한 현상이다.

5.7 Layer 관점에서의 재해석 

기존 설명에서는 XOR 체커보드를 XOR 함수의 특수한 시각적 특징으로 이해하는 경우가 많았다.

그러나 본 연구는 이를 보다 일반적인 Layer 구조의 관점에서 해석한다.

체커보드는 XOR 자체에서 직접 발생하는 것이 아니라

Layer 선택 구조 Gray Code 배열의 인접성 보존 성질 

이 결합되어 나타나는 결과이다.

따라서 체커보드는 Boolean 공간의 구조가 2차원 카르노맵 위에 표현된 사례로 볼 수 있다.

5.8 소결론 

XOR 및 XNOR 함수의 체커보드 패턴은 단순한 시각적 특징이 아니다.

이는 Boolean 공간의 Layer 구조와 Gray Code 배열의 인접성 보존 성질이 결합되어 나타나는 구조적 현상이다.

본 연구의 결과는 카르노맵이 단순한 논리 최소화 도구를 넘어 Boolean 공간 구조를 시각적으로 표현하는 도구로 해석될 수 있음을 시사한다.
